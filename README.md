AI Email Auto Responder using n8n, Google Gemini & Gmail Automation

An AI-powered email automation workflow built with n8n, Google Gemini (PaLM API), Gmail API, and Airtable.

This project automatically reads incoming Gmail messages, generates intelligent AI-based replies using a large language model, logs the interaction in Airtable, and sends automated responses — creating a complete AI email assistant system.

Project Overview

This AI Auto Responder is a fully automated workflow designed to:
Monitor Gmail inbox on a scheduled trigger
Fetch recent emails
Use Google Gemini AI to generate contextual replies
Store sender details and AI-generated drafts in Airtable
Automatically send the response via Gmail
Maintain workflow visibility with email state handling
It demonstrates real-world AI integration in business process automation using low-code tools.

How It Works
Schedule Trigger → Fetch Emails → AI Reply Generation → Store in Airtable → Send Email → Update Email Status
The system leverages:
n8n Schedule Trigger for automation
Gmail OAuth2 for secure email access
LangChain Agent Node for AI orchestration
Google Gemini Chat Model for natural language generation
Airtable API for structured logging
